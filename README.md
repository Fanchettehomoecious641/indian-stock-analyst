## 🧩 What this does

`indian-stock-analyst` turns your Zerodha or Groww portfolio CSV into a clean stock report.

It reads your holdings file and builds a simple Buy, Hold, or Avoid view for each stock. You can use it to review your portfolio without digging through charts, ratios, or market noise.

This tool is built for Windows users who want a fast way to check their stock list and get a plain report they can read at a glance.

## 💻 Before you start

You need:

- A Windows PC
- Your Zerodha or Groww portfolio CSV file
- A web browser to open the download page
- Enough space to save the app and report files

Best results come from a recent Windows 10 or Windows 11 system.

Before you run the app, make sure your portfolio file is ready.

For Zerodha, export your holdings or portfolio data as CSV.
